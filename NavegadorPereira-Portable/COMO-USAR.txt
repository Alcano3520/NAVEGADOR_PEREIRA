═══════════════════════════════════════════════════════════
  HACER PORTABLE (USB, Disco Externo)
═══════════════════════════════════════════════════════════

1. Copia TODA la carpeta "NavegadorPereira-Portable"
2. Pégala donde quieras (USB, otro disco, etc.)
3. Ejecuta Pereira.bat desde la nueva ubicación

¡Listo! Todo funciona igual desde cualquier lugar.


═══════════════════════════════════════════════════════════
  PERSONALIZACION
═══════════════════════════════════════════════════════════

Editar Configuraciones de Privacidad:
→ Abre: config\preferences.json
→ Modifica los valores (true/false, 1/2)
→ Guarda y reinicia el navegador

Bloquear Más Dominios:
→ Abre: config\blocked-domains.txt
→ Añade dominios (uno por línea)
→ Guarda

Cambiar Flags de Chrome:
→ Abre: config\flags.txt
→ Añade/elimina flags
→ Guarda y reinicia

═══════════════════════════════════════════════════════════
  INSTALAR EXTENSIONES
═══════════════════════════════════════════════════════════

1. Abre el navegador (Pereira.bat)
2. Ve a: chrome://extensions/
3. Activa "Modo de desarrollador" (esquina superior derecha)
4. Arrastra archivo .crx de la extensión
   O instala desde Chrome Web Store

Extensiones Recomendadas para Privacidad:
- uBlock Origin (bloqueador de anuncios/rastreadores)
- Privacy Badger (anti-rastreo)
- HTTPS Everywhere (forzar HTTPS)
- ClearURLs (limpiar parámetros de rastreo)
- Decentraleyes (CDN local)


═══════════════════════════════════════════════════════════
  VERIFICAR PRIVACIDAD
═══════════════════════════════════════════════════════════

1. Abre el navegador
2. Ve a: chrome://settings/privacy
3. Verifica que:
   ✓ Sincronización: Desactivada
   ✓ Cookies de terceros: Bloqueadas
   ✓ Do Not Track: Activado
   ✓ Precarga de páginas: Desactivado

4. Prueba en: https://browserleaks.com/
   - Verifica qué información filtra tu navegador


═══════════════════════════════════════════════════════════
  MOTORES DE BUSQUEDA PRIVADOS
═══════════════════════════════════════════════════════════

Recomendados:

═══════════════════════════════════════════════════════════
  PROBLEMAS COMUNES
═══════════════════════════════════════════════════════════

P: No se abre el navegador
R: Ejecuta INSTALAR.bat primero

P: Error al actualizar
R: Cierra el navegador completamente y reintenta

P: Perdí mis marcadores
R: Están en profile\Default\Bookmarks (copia de seguridad)

P: No funciona en otra PC
R: Copia TODA la carpeta NavegadorPereira-Portable

P: Página web no funciona
R: Algunos sitios requieren cookies de terceros
   Permite temporalmente para ese sitio en chrome://settings/content

P: Videos no se reproducen
R: El navegador incluye codecs. Si falla, prueba otra página
